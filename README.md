# scrape_friend_list

This is an Application to scrape your friend's list using Beautiful Soup library.</br>
Mostly making a list of all your friend's on facebook is a manual process which is quite cumbersome. This is an automation of the same. </br>
Add the dummy variables in their places like 'your_email', 'your_password' and 'url'.</br>
Install 'lxml' parser by pip install lxml (though you can use 'html-parser' as well). </br>
Run using:  python3 scrape_friends.py
